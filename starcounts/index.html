Catalog Query Approach

Catalog: Gaia DR3

~1.8 billion sources with G, GBP, GRP photometry to G≈21. Complete for guide-star brightness range.

TAP Query

Each sensor FOV is queried via ADQL cone search. Primary endpoint (ESA), fallback (VizieR), final fallback (Poisson simulation).

POST https://gea.esac.esa.int/tap-server/tap/sync
  REQUEST=doQuery & LANG=ADQL & FORMAT=csv
SELECT TOP 2000 ra, dec, l, b,
       phot_g_mean_mag  AS gmag,
       phot_bp_mean_mag AS bpmag,
       phot_rp_mean_mag AS rpmag
FROM gaiadr3.gaia_source
WHERE 1=CONTAINS(
    POINT('ICRS', ra, dec),
    CIRCLE('ICRS', <ra>, <dec>, <radius>))
  AND phot_g_mean_mag BETWEEN <gMin> AND <gMax>

Search radius = FOV diagonal / 2 × 1.1. Galactic → equatorial via IAU rotation matrix.

Band Conversion

Gaia G/BP/RP → target band via polynomial color relations (c = GBP−GRP). Query range padded ±1.5 mag for color spread.

G − V = −0.027 + 0.014c − 0.216c² + 0.014c³
G − r' = −0.129 + 0.247c − 0.027c² − 0.049c³

Johnson B, Rc, Ic and SDSS u', g', i', z' derived similarly (Riello+ 2021, Evans+ 2018). Typical σ≈0.05–0.15 mag.

Star Count Model

Cumulative pole counts N(r') from Bahcall & Soneira (1980) + SDSS/Gaia calibration. Latitude scaling:

N(m, l, b) = Ndisk(pole) × csc|b| × (1 + bulge) + Nsph

Other bands mapped via color offset: N(<m, X) = Nr'(m + Δ), where Δ is the typical (r'−X) color.
